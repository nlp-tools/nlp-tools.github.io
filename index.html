NLP Tools

Useful Tools for Natural Language Processing

Photo found at Unsplash.

This website contains several applications designed for anyone undertaking natural language processing. Currently we offer two different types of tools: annotation tools, made for annotating and creating data for use in machine learning, and visualisation tools, made for visualising the results of various NLP tasks.
All software on this website is open source and freely available for download via GitHub.

Annotation Tools

  • Redcoat - Collaborative Annotation Tool

    Redcoat is a collaborative annotation tool for labelling entity recognition data. Its keyboard-based interface allows for very rapid annotation. It is designed to be extremely easy to set up and distribute to multiple annotators.

  • Lexnorm-Labeller

    Lexnorm-Labeller is a simple tool that can be used to annotate data for lexical normalisation. Like Redcoat, it is purely keyboard-based to facilitate rapid annotation.

Visualisation Tools

  • Lexnorm-Vis (Work in Progress)

    Lexnorm-Vis is a simple web-based platform that visualises the results of lexical normalisation tasks. The platform also provides statistics such as F1-Score, total correct/incorrect, and the F1-Score per unique token.

  • ER Graph (Work in Progress)

    ER (Entity Recognition) Graph visualises the links between entities in a dataset. Domain experts may interact with the visualisation to quickly discover links between entities that were previously obscured by the sheer volume of text in the original dataset.
