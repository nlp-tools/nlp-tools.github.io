This website contains several applications designed for anyone undertaking natural language processing. All software on this website is open source and freely available for download via GitHub.

Tools Available for Download

  • CoAT

    CoAT is a collaborative annotation tool for labelling entity recognition data. Its keyboard-based interface allows for extremely rapid annotation.

Works in Progress

  • Lexnorm-Vis

    Lexnorm-Vis is a simple web-based platform that visualises the results of lexical normalisation tasks.

  • ER Graph

    ER (Entity Recognition) Graph visualises the links between entities. This allows domain experts to easy view the relationships between entities in the data.
