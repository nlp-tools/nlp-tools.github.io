Redcoat is a collaborative annotation tool for labelling entity recognition data. Its keyboard-based interface allows for very rapid annotation. It is designed to be extremely easy to set up and distribute to multiple annotators.

Redcoat has three primary goals:

  1. Enable faster annotation than is currently possible with other annotation tools;
  2. Maximise the simplicity of setting up an annotation project; and
  3. Make it easy to collaborate and distribute the project to annotators.

Annotation Speed

One key feature that differentiates Redcoat from other annotation tools is its keyboard-based interface. Rather than selecting tokens with the mouse, annotators use the arrow keys to navigate between tokens, and the number keys to label entities as a certain class. Annotation becomes much quicker as tagging each entity requires one key press rather than mouse movement, selection, and clicking.

Project Setup

Many annotation tools require the user to be familiar with the file structure requirements of the tool before an annotation project can be released. Setting up a project therefore requires a great deal of time spent reading documentation and ensuring the files are in the correct format.

Redcoat's simple project setup page prompts the user for all details relating to the project, bypassing this problem. The project can be setup regardless of whether your data is tokenized, and listing the entity categories can be done through the web rather than through a carefully-formatted file if desired.

Distributing Projects

Most existing tools require the project creator to manually divide their data into a number of folders and send a link to each annotator. Just like setting up a project, this also requires a lot of time to set up correctly.

Redcoat can divide your data into a certain number of batches automatically. It will provide you with a list of URLs that you may email to the annotators yourself. Alternatively, you may elect to automatically email the links out to the annotators using the built-in mail service, saving vast amounts of your time.
