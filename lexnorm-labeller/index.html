Lexnorm-Labeller

Simple Annotation Tool for Lexical Normalisation

Background generated with Trianglify-Generator.

Lexnorm-Labeller is a simple tool for annotating lexical normalisation data. It is completely keyboard-based, allowing users to annotate data quicker than with mouse-based tools. It is available on GitHub.

Overview

Lexnorm-Labeller is powered by a small Python script that generates JSON, which is displayed in a simple web interface. Users may annotate their data by cycling through the tokens using the Tab key, and typing in the correct form of each token. Tokens that are already correct can be quickly annotated by pressing Tab.

When annotation is complete, the web tool generates a JSON file. Each record in this file contains an "index" (a number representing the record's index in the dataset), "input" (the data prior to normalisation), and "output" (the data after normalisation). This format is the same as the format used for the lexical normalisation task in the Workshop on Noisy User-Generated Text 2015 (W-NUT).

Users can specify a list of domain-specific terms and acronyms that will be ignored during the generation of the JSON data. At this stage, Lexnorm-Labeller only allows for the annotation of tokens that are non-standard words, which do not appear in the list of domain-specific terms or acronyms. In future it will be possible to annotate all tokens in the data, cycling through already-correct tokens using the Tab key.
